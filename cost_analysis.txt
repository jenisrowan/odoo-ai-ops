================================================================================
                     MONTHLY COST ANALYSIS — AWS INFRASTRUCTURE
                     Odoo AI Ops (us-east-1 Region Pricing)
================================================================================

Assumptions:
  • All prices in USD/month unless stated otherwise.
  • EC2 pricing uses On-Demand rates (no Reserved Instances or Savings Plans).
  • Odoo uses m7i-flex.large (2 vCPU, 8 GiB RAM).
  • Shared App Cluster and ClickHouse use t3.small (2 vCPU, 2 GiB RAM).
  • RDS PostgreSQL uses db.t4g.small (2 vCPU, 2 GiB RAM) Multi-AZ.
  • HA model: 1 active instance per service. 2nd instance boots on demand
    via ECS auto-scaling (not always-on) for Odoo and Shared App only.
  • VPC Endpoints: Gateway for S3 (free). No interface endpoints.
  • NAT Gateway is used for external API egress and SQS polling.
  • "Idle" = infrastructure running with zero order volume.
  • "Low" = ~100 flagged orders/day (webhook-triggered AI tasks).
  • "Growth" = ~1,000 flagged orders/day.
  • Order Risk Profile (of flagged orders):
      - No/Very Low Risk orders are bypassed by Odoo (0 AI tokens used).
      - 70% of flagged orders are Medium Risk (Claude 4.5 Haiku).
      - 30% of flagged orders are High Risk (Claude 4.6 Sonnet).

---  SECTION 3: SCALING PROJECTIONS  ------------------------------------------

Assumptions per flagged order:
  • 1 SQS message received + 1 SQS delete = 2 SQS requests
  • 1 API Gateway invocation (webhook ingestion)
  • 1 Lambda Authorizer invocation
  • 1 Shopify GraphQL call (~20 KB response) → via NAT

---  SECTION 5: COST RISK FLAGS  ----------------------------------------------

  ⚠ db.t4g.small CPU CREDIT DEPLETION
    The db.t4g.small is a burstable instance. Under sustained heavy
    queries (e.g., Langfuse writing trace data while Odoo polls actively),
    CPU credits can deplete, throttling database performance to baseline.
    Monitor CPUCreditBalance in CloudWatch. If sustained < 20 credits,
    consider enabling Unlimited Mode ($0.05/vCPU-hr for surplus credits).

  ⚠ t3.small MEMORY CEILING (Shared App Cluster)
    The t3.small has only 2 GiB RAM shared across PgBouncer and FastAPI.
    Heavy LangGraph workloads could still cause OOM kills. Monitor ECS MemoryUtilization.
